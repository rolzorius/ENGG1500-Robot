The files in this folder are all required for the successful operation of the robot using a PICO-W.

Ensure they are all flashed to the bot.

VariConPins.py contains all the necessary variables, constants, and pin numbers for the main.py file. If you 
need to change anything on the main.py please ensure you edit the VariConPins.py file instead of changing the
variables in the main.py.
